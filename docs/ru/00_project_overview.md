# Обзор проекта

Проект демонстрирует организацию Secure SDLC и DevSecOps workflow в Qubes OS.

Цель проекта — показать практический цикл:

- создание intentionally vulnerable приложения
- запуск автоматизированных security scans
- фиксация findings
- блокировка небезопасной версии через policy gate
- remediation
- повторная проверка
- успешное прохождение policy gate
- сбор evidence
- подготовка отчёта и презентации
