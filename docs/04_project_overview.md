# Project Overview / Обзор проекта

## RU

Проект демонстрирует организацию Secure SDLC и DevSecOps workflow в Qubes OS.

Цель — показать практический цикл:

- создание intentionally vulnerable приложения;
- автоматизированные security scans;
- фиксация findings;
- блокировка небезопасной версии через policy gate;
- remediation;
- повторная проверка;
- успешное прохождение policy gate;
- сбор evidence;
- подготовка отчёта и презентации.

## EN

The project demonstrates a Secure SDLC and DevSecOps workflow in Qubes OS.

The goal is to show a practical cycle:

- building an intentionally vulnerable application;
- running automated security scans;
- recording findings;
- blocking the unsafe version through a policy gate;
- remediation;
- repeated validation;
- successful policy gate pass;
- evidence collection;
- report and presentation preparation.
