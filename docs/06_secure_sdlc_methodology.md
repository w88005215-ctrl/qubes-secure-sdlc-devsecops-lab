# Secure SDLC Methodology / Методология Secure SDLC

## RU

Методологическая база проекта:

- OWASP ASVS;
- OWASP SAMM;
- NIST SSDF;
- SLSA;
- CycloneDX.

Стандарты используются как практическая опора для проектирования pipeline, security gates, SBOM и remediation workflow.

## EN

The project methodology is based on:

- OWASP ASVS;
- OWASP SAMM;
- NIST SSDF;
- SLSA;
- CycloneDX.

These standards are used as practical guidance for pipeline design, security gates, SBOM and remediation workflow.
