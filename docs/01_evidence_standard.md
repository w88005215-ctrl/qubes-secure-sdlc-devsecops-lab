# Evidence Standard / Стандарт evidence

## RU

Все ключевые действия проекта должны оставлять воспроизводимые доказательства в каталоге evidence/.

Стандарт именования:

DEVSECOPS_STEP_TYPE_DESCRIPTION.ext

Типы evidence:

| Type | Пример |
|---|---|
| OUTPUT | вывод команды |
| REPORT | отчёт security tool |
| SBOM | Software Bill of Materials |
| SCREENSHOT | скриншот |
| DIFF | сравнение before/after |

## EN

Every key project action must produce reproducible evidence under evidence/.

Naming standard:

DEVSECOPS_STEP_TYPE_DESCRIPTION.ext

Evidence types:

| Type | Example |
|---|---|
| OUTPUT | command output |
| REPORT | security tool report |
| SBOM | Software Bill of Materials |
| SCREENSHOT | screenshot |
| DIFF | before/after comparison |
