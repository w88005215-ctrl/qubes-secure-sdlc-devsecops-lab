# Git Branching Strategy / Стратегия веток Git

## RU

Проект использует простую портфельную стратегию веток.

| Branch | Назначение |
|---|---|
| main | стабильная финальная версия |
| develop | интеграционная рабочая ветка |
| vulnerable-baseline | состояние intentionally vulnerable приложения |
| remediation | исправления findings |
| evidence-pack | подготовка evidence artifacts |

Планируемые теги:

- v0.1-skeleton
- v1.0-vulnerable-baseline
- v2.0-fixed-gate-passed

## EN

The project uses a simple portfolio-friendly branching strategy.

| Branch | Purpose |
|---|---|
| main | stable final version |
| develop | integration branch |
| vulnerable-baseline | intentionally vulnerable application state |
| remediation | findings remediation |
| evidence-pack | evidence artifacts preparation |

Planned tags:
