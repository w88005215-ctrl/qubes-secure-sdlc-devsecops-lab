# Qubes Architecture / Архитектура Qubes

## RU

| Qube | Назначение |
|---|---|
| dev-workbench | разработка репозитория и demo app |
| security-runner | запуск security tools |
| evidence-vault | хранение evidence |
| report-vm | отчёты и презентация |
| ai-workbench | опциональный local AI для sanitized findings |

dom0 не используется для рабочих файлов проекта.
